Damon & Felix Die-Cast Racing League

  • Date: December 2022/January 2023
  • URL: peterjungers.com/dfdrl/index
  • Code: github.com/peterjungers/dfdrl
  • Learning objectives: Create a sizeable site using the Flask framework; design SQLite database and improve SQLAlchemy query skills; utilize Bootstrap for CSS styling; use custom JavaScript to create searchable vehicle table.
  • Unforeseen learning outcomes: Better understanding of the Jinja2 templating language and its capabilities; more familiarity with Bootstrap source file; less use of constructing raw SQL queries as a precursor to constructing SQLAlchemy queriesâ€”most queries were constructed with SQLAlchemy from the beginning.

The Damon & Felix Die-Cast Racing League, of which I am a board member, team manager, and lead statistician, is a high-performance die-cast racing league, comprised primarily of Hot Wheels vehicles, but with a healthy showing from Matchbox. Each season culminates in the Champions League, where twelve vehicles compete to be the league champion. More information about the workings of the league are found on the site itself.

I have been keeping stats of various kinds for the league since its first season, primarily using Google Sheets, which helped inspire the look and functionality of the new site. My other inspiration came from my first Python project, which was a searchable database that directly inspired the vehicles stats search on the new site. My primary goal in creating the site was not only to have a historical record of the league, but to create functionality for team managers to better track their vehicles.

Mario Kart 8 Deluxe Cup Generator

  • Date: October 2022
  • URL: peterjungers.com/mk8d/index
  • Code: github.com/peterjungers/mk8d_project
  • Learning objectives: Improve Flask skills; improve HTML and custom CSS coding know-how; design SQLite database and improve SQLAlchemy query skills.
  • Unforeseen learning outcomes: A more solid understanding of and appreciation for the CSS grid property.

Mario Kart 8 Deluxe for the Nintendo Switch contains, to date, 72 tracks spread over 18 cups. The goal of my Cup Generator is to expand upon these original cups by constructing new ones based on thematic similarities across tracks throughout the game. For example, a number of tracks are set in the sky; others take place within large buildings. I therefore created a database of the tracks in the game, determined what the new thematic cups would be, then constructed the user interface to allow the easy viewing of these new cups. Like with the DFDRL project, I wanted to create a site that I could use. The site is updated and expanded every few months as Nintendo releases new tracks for the game.

This was my second Flask project, and first that was completely independent. My first Flask site, though necessarily containing unique SQLAlchemy queries, relied heavily on tutorials for its HTML and CSS. With the Mario Kart 8 Deluxe Cup Generator, however, I became more comfortable and confident with all aspects of the framework, a trend that continues with each subsequent project I work on..
